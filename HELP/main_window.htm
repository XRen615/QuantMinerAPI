Main Window



QuantMiner starts with a main window and a menu that gives access to all the functionalities. The menu is as follows:


File menu:

Open File: allows choosing a dataset to analyze. A dataset should be in a DBF (dBase version 4) format or CSV Comma Separated Value File Format. In order to know more about the expected form of the table, please go to section association rules.
Once the dataset selected, QuantMiner starts a 5-step wizard that guides you through association rule mining.

Close (initially in gray): Allows to end a session on this dataset.
Note that to save your current state of experiments, you need to use the profiles menu for parameters setting and the 3 rule file formats for the results. Thus, closing a dataset does not save your current work.

Exit: Closes QuantMiner.
As noted in the previous menu, closing QuantMiner does not save your work unless you save your profile and results.


Profiles (initially in gray):

Load a profile: Allows loading a profile of any configuration of the first 3 steps of the wizard.

Save the current profile: Allows saving a profile of any configuration of the first 3 steps of the wizard to be reused in other sessions.
See profiles to know more about this important functionality.


Tools:

Get information on a rule file: it is advised to give meaningful names to the files storing your results. The command allows to analyze and provides some information about a rule file saved in QuantMiner format.

QuantMiner parameters: Allows to open the parameter setting command to specify user preferences.


Help:

Help contents: Allows to browse QuantMiner help.

About QuantMiner...: Some additional information about the system development.
